练习：打印100-999中十位数上不是3的所有的数

#遍历整数列表[101,102,103,104,105,106,……,999] 访问的值赋值给x
for x in range(100,1000):
    #判断一个三位数的十位数是否等于3，如果是执行循环体：continue。否则不执行
    if x // 10 % 10 == 3:
        #程序执行遇到continue就不执行循环体下面的内容，直接进入下一次循环的判断
        continue
    print(x,end=' ')
