#6运算符
python中的运算符:算术运算符、比较运算符、赋值运算符、逻辑运算符

##6.1算术运算符
算术运算符:+(加)、-(减)、(乘)、/(除)、%(求余)、//(整除)、**(幂运算)
+、-、、/、%和数学里面相应的运算时一模一样的
代码如下：
print(12+10.0) #22.0
print(100-8) #92
print(10*7) #70
print(5/2) #2.5
print(10%4) #2
运行结果如下：
22.0
92
70
2.5
2
// (整除)：可以对浮点数做整除的操作，只是结果只保留整数的部分(直接去掉小数部分)
print(10.0//3) #3.0
print(16/10) #1.6
print(16//10.0) #1.0
print(10//4) #2
计算次方
print(23) #计算2的3次方
练习：获取n的各位数、十位数和百位数(要求n,大于100)
n = 7896
取个位数: (对10取余)
print(‘个数:’,n%10)
十位数：
print(‘十位数:’,n//10%10)
print(‘十位数:’,n%100//10)
百位数:
print(‘百位数:’,n//100%10)
print(‘百位数:’,n%1000//100)
运行结果：
个数: 4
十位数: 8
十位数: 8
百位数: 7
百位数: 7

##6.2比较运算符

  • 比较运算符：>、<、>=、<=、==、!=
  • 所有比较运算符的结果都是布尔值

  • x>y :比较x是否大于y,如果是结果就是True，否则结果是False 

    print(10 > 5)   #True
print(6 > 10.8) #False
print('abcdef' > 'bcd') #False
#比较字符串的大小的时候，不是比较两个字符串的长度
#而是从两个字符串的第一个字符开始比较，如果第一个字符串的第
#一个符大于第二个字符串的字一个字符，结果就是true,否则false。
#如果两个字符串的第一个字符相等才去比较他们的第二个字符，依次类推
print('abc'>'aba')
print('abc'>'abcd')
print('abc'>'abc')

  • x < y:比较x是否小于y。如果是，结果就是True，否则结果是False 

    print(10 < 5)
print(6 < 10.8)
print('abc' < 'aba')

  • x>=y :比较x是否大于或者等于y。x大于y或者x等于y,结果是Ture；否则是False 

    print(10 >= 5)
print(10 >= 10)
print('abc' >= 'abc')

  • x<=y :比较x是否小于或者等于y。x小于y或者x等于y,结果是True；否则是False 

    print(10 <= 5)
print(10 <= 10)

  • x==y :比较x和y是否相等. 如果相等就是True， 否则结果是False 

    a = 10
b = 20
print(a == 10)
print(a == b)

  • x != y :比较和y是否不相等。如果不相等就是True,否则结果是False 

    print(a != 10)
print(a != b)

    6.3.赋值运算符: = 、 += 、-=、*= 、/=、%=、//=、**=

  • 所有的赋值运算符左边都必须是变量，右边是表达式
    a = 20
b = a  #如果a的值是基本数据类型(数字、字符串)，直接将a里面存的值赋给b；如果值是对象，传递的是变量中存储的地址
a = 30

  • 表达式（本身有值的语句）： 10，a, 10+a、10>20、10+20/4*3等

    # a + 10 = 10  # SyntaxError

  • =

    num = a + 30

  • += 、-=、*= 、/=、%=、//=、**= 

    a + 10
print(a)
a += 10  # a = a + 10
a += 1   # a = a + 1
print(a)
a -= 10  # a = a - 10
print(a)
a *= 2   # a = a * 2
print(a)
a /= 2   # a = a / 2
print(a)
a //= 2  # a = a // 2
print(a)   
a **= 2  # a = a**2
print(a)

6.4逻辑运算符: and(与), or(或), not(非)

  • 逻辑运算符中表达式的值都是布尔值

    # 表达式1 and 表达式2 ：两个表达式的值都为True，结果才是True，否则是False 。(and -> 并且)
print(True and True)
print(True and False)
print(False and True)
print(False and False)

age = 20
face = 90
  • 进网吧的要求要求:年龄大于18并且小于50,颜值还不能低于80
    # age > 18
# age < 50
result = age > 18 and age < 50 and face >= 80
print(result)

  • 表达式1 or 表达式2 ：两个表达式的值都为False,结果才是False, 否则是True。 (or -> 或者) 

    print(True or True)
print(True or False)
print(False or True)
print(False or False)

grade = 9.0
score = 70

  • 奖学金要求:绩点大于8.5 或者操评大于90

    result = grade > 8.5 or score > 90
print(result)

  • 成绩在8.5-9.7之间

    grade >= 8.5 and grade <= 9.7
8.5 <= grade <= 9.7

  • 奖学金新要求:1.绩点大于9.7 2.成绩8.5-9.7，操评分大于85

  • 运算表达式中，加()可以改变运算顺序
    result = grade > 9.7 or (8.5<=grade<=9.7 and score > 85)

  • not 表达式 : 如果表示式的值为True，结果就是False;表达式的值为False，结果就是True 

    print(not True)
print(not False)

  • 及格的要求：成绩不小于6

    grade = 5  #给变量赋值的时候，后面赋的值会覆盖前的值
result = not (grade < 6)
print(result)
  • 进入游乐场年龄的要求:不大于12岁并且不小于2岁
    age = 20 
result = (not age > 12) and (not age < 2)
print(result)

    6.5运算符优先级

  • 赋值运算符 < 逻辑运算符 < 比较运算符 < 加减 < 乘、除、取余、整除、幂运算 < +(正号)、-(负号)
  • 可以加括号改变运算的运算顺序（遇到括号就先算括号里面的）
    a = True and False
print(a)
result = -10 and 2
print(result)
result = 0 > -10 and 2 < 5
print(result)
result = 2 + 10 > 3
print(result)

7.1python中的字符串都是Unicode字符串

  • Unicode编码：就是一种对字符的编码方式(将字符编码成对应的数字，方便计算机存储)
  • Unicode编码又叫万国码，支持目前几乎所有的语言文字编码
  • Unicode包含ASCII码

  • 数据存储到计算机中是以2进制的形式存的（二进制是数字）

  • 将Unicode码转换成字符 

    char = chr(0x4e01)
print(char)

  • 将字符转换成Unicode码(结果是10进制形式)

    code = ord('婷')
print(code)
print(chr(20313))

for x in range(0x2C80,0x2CFF):
    print(chr(x), end=' ')
print()

  • 字符串比较大小的时候，实质就是比字符的Unicode编码的大小

    str1 = '你好，Python'
length = len(str1)
print(length)

7.3获取字符/子串

str1 = 'Hello,Python'
  • a.获取某一个字符
  • 格式: 字符串变量[下标]
  • 下标: 从0开始的数字，代表的是某一个字符在字符串中偏移量(位置)（范围：0 ~ 字符串长度-1;）

  • 注意: 下标不要越界，否则会报IndexError错误

    print(str1[0])  # 获取第0个字符
print(str1[5])    # 获取从0开始数的第5个字符  
print(str1[20]) # IndexError: string index out of range(下标越界)

  • 下标也可以是负数, 负的下标也不可以越界

    print(str1[-1]) # 获取的是倒数第一个字符(最后一个)
print(str1[-2])    # 获取倒数第二个字符
print(str1[-13]) # IndexError: string index out of range  

print(str1[len(str1)-1])  # 获取最后一个字符
  • b.获取字符串中某一部分的字母(获取子串)

  • 格式: 字符串变量[开始下标:结束下标] —> 获取从开始下标到结束下标前的所有字符

    str1 = 'Hello,Python!'

  • 注意:开始下标对应的字符可以取到，结束下标对应的字符是取不到的

    print(str1[1:4])  #获取从下标是1开始，到下标是(4-1)的所有的字符
print(str1[6:10]) #获取从下标是6开始，到下标是9的所有的字符

  • 开始下标不写，默认就是0

    print(str1[:5])   #获取从开始到下标是4为止的所有的字符

  • 结束下标不写，就会取到最后一个字符

    print(str1[-4:])

print(str1[:])

  • 如果结束下标在开始下标的前面，就会取不到字符(不能倒着取)

    print(str1[5:1])

  • + : 字符串中的+操作，就是字符串连接 

    str2 = 'aaa' + 'bcd'
print(str2)
str1 = 'Hello'

str3 = str1 + '\t' + str2
print(str3)
str2 += '!'   # str2 = str2 + '!'
print(str2)

  • * : 字符串中的*操作，就是字符串重复多少次

    str1 = 'abc' * 3  # abc重复3次
print(str1)

  • 字符串1 in 字符串2: 判断字符串2中是否包含字符1 

    str1 = 'Hello'
print('He' in str1)  # 判断'He'是否在str1里面(判断str1中是否包含'He')

  • 字符串1 not in 字符串2： 判断字符串1是否不在字符串2中 

    print('ae' not in str1)

8.1.格式化输出

  • print(‘%s %d %c’%(var1, var2, var3)) : 输出字符串的时候，在字符串中使用字符串格式符表示变化的内容。#然后在%后面的括号里面，依次使用表达式给前面的字符串格式符赋值

  • %s: 字符串 %d:整数 %f:浮点数 %c:字符 

    name = '张三'
age = 28
# 我是XXX,今年XX岁
print('我是%s,今年%d岁'%(name,age))
# %f
print('余额:%f万元'%(10.25))
# %.2f 保留小数点的后两位   %.3f 保留小数点后三位
print('余额:%.2f万元'%(10.25))
python中的字符，就是指长度是1的字符串
print('%c'%('a'))  # 打印的是 a 
print('%c'%(97))   # 打印的是 a
print('%c'%(0x4e00))  #打印的是 一
# %o 
print('%o'%(10))
# %X/%x
print('%x , %X'%(15,15))

##8.2.设置print函数的sep参数，可以改变print函数在同时打印多个内容时的分割符(默认是空格)
name = ‘张三’
age = 28

- 我是XXX,今年XX岁

  • 注意:加号两边只能都是数字或者都是字符串，不能一个数字一个字符串

    # print('我是'+name+",今年"+age+'岁')
print('我是',name,",今年",age,'岁',sep='')  # 打印多个数据的时候，数据之前没有间隙

print('aa',100,'bbb',sep='--')   # 打印多个数据的时候，数据之间以‘--’隔开

##8.3设置print函数的结束字符串，默认是’\n’
print(‘Hello’,end=’ ‘)
print(‘World’)
print(‘aaa’)

  • 注意: 设置sep和end参数，只是当次有效

#9.字符串的内置函数

9.1capitalize() 将字符串的第一个字符转换为大写 (不会改变原来的字符串，而是返回一个新的字符串)

str1 = ‘python student’
newStr = str1.capitalize()
print(str1, newStr)

2.title() 将字符串中每个单词的首字母变成大写(不会改变原来的字符串，而是返回一个新的字符串)

单词的区分，和英语区分单词的方式是一样的(以空格或者标点符号分开的字符串)

newStr = str1.title()
print(str1, newStr)

#how are you?I am fine,thank you.and you?

3. center(width, fillchar) 将字符串变成指定的长度，并且原字符串内容居中，剩余的位置使用指定的字符fillchar填充

str1 = 'abc'
newStr = str1.center(11,'*')
print(newStr)
